# Flutter Lessons

## Getting Started

This project is a starting point for learn the basics of the Flutter framework.

- Navigation with GoRouter
- State Management with Riverpod
- Widgets Composition
- Basic Widgets
- Asynchronous Operations
- Domain Driven Design
- Material Design 3
- Forms Management
- Theming and Styling
- Code Generation
